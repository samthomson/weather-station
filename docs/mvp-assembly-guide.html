Weather Station — Stripboard Build

30-pin ESP32 · 6×9 cm stripboard · rows 1–8 from top

Stripboard matrix

One row = one net. Solder each pin into any hole on that row. Male dupont leg through hole, bend on copper side. ESP32 off-board (F on ESP, M soldered).
