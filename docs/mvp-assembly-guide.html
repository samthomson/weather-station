Breadboard setup

ESP32 sits off the breadboard. Jumper wires connect it to the breadboard.

Rails — the long rows along the edges marked + and −. Every hole in a rail is connected to every other hole in that same rail. Pick any hole.
Columns — in the main grid, each column number (e.g. column 5) has 5 holes: A5, B5, C5, D5, E5. All 5 are connected together. Pick any free hole in that column.

We use:
Top + rail = 3.3V power
Top − rail = ground
Column 5 (holes A5–E5) = SDA
Column 10 (holes A10–E10) = SCL
Column 15 (holes A15–E15) = PMS TX
Column 20 (holes A20–E20) = PMS RX
Column 25 (holes A25–E25) = PMS 5V
Column 30 (holes A30–E30) = Rain AO
